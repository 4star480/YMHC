# YMHC — Youth Mental Health Collaborative Africa

Modern NGO website built from WhatsApp brief + reference sites (Mental Health Africa, StrongMinds, Pamoja-style NGOs).

## Open locally

Double-click **`index.html`** or run:

```bash
cd ymhc-africa
python -m http.server 8080
```

Then open http://localhost:8080

## Features (vs. reference sites)

- Hero with impact stats, crisis helpline bar, WhatsApp float
- **Research · Capacity building · Community outreaches** (full program copy)
- Multi-step booking / partnership form
- Events, testimonials slider, resource library
- FAQ, team placeholders, partners, newsletter
- Volunteer & partner modals
- Mobile navigation, accessibility skip link
